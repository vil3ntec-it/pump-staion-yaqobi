# برنامهٔ کامپیوتری پمپ یعقوبی (نسخهٔ دسکتاپ)

این پوشه، خودِ برنامهٔ پمپ یعقوبی را به یک **برنامهٔ نصب‌شدنی ویندوز** تبدیل می‌کند:

- یک نصب‌کنندهٔ `.exe` با ویزارد نصب — هنگام نصب می‌توانید **درایو و مسیر نصب را انتخاب کنید** (مثلاً `D:\`).
- **صفحهٔ لودینگ (اسپلش)** هنگام باز شدن، مثل نرم‌افزارهای بزرگ.
- اجرای کاملاً **آفلاین و محلی** — بدون نیاز به اینترنت یا سرور.
- ذخیرهٔ داده روی یک **پروتکل امنِ اختصاصی (`app://`)** تا سهمیهٔ ذخیره‌سازی بر اساس فضای دیسک باشد، نه سقف ۵ مگابایتیِ تبِ مرورگر.
- **سرور خانگیِ جاسازی‌شده:** از منوی «سرور خانگی › باز کردن پنل سرور» می‌توانی با یک کلیک این کامپیوتر را به سرورِ مرکزی تبدیل کنی و آدرس + رمز را به بقیهٔ دستگاه‌ها بدهی — بدون نیاز به zip و فایل bat جداگانه. داده در پوشهٔ قابل‌نوشتنِ کاربر (`userData/server-data`) ذخیره می‌شود و رمز سرور ثابت می‌ماند.

## چطور فایل نصب را بگیرم؟

نیازی به کامپیوتر برنامه‌نویس نیست — گیت‌هاب خودش می‌سازد:

1. در گیت‌هاب وارد تب **Actions** شوید.
2. ورک‌فلوی **«ساخت برنامهٔ کامپیوتری (ویندوز)»** را باز کنید و دکمهٔ **Run workflow** را بزنید.
3. چند دقیقه صبر کنید تا سبز شود. سپس فایل نصب در دو جا هست:
   - بخش **Releases** ریپو (نام: «برنامهٔ کامپیوتری پمپ یعقوبی») — فایل `PumpYaqobi-Setup-*.exe`.
   - یا در همان صفحهٔ اجرای ورک‌فلو، بخش **Artifacts**.
4. فایل `.exe` را دانلود و روی ویندوز اجرا کنید. (بار اول ویندوز SmartScreen ممکن است هشدار بدهد چون فایل امضای دیجیتال ندارد — روی «More info → Run anyway» بزنید.)

## ساخت دستی روی کامپیوتر خودتان (اختیاری)

اگر Node.js نصب دارید:

```bash
cd desktop
npm install
npm run dist:win      # نصب‌کنندهٔ ویندوز در پوشهٔ desktop/dist ساخته می‌شود
# یا برای تست سریع بدون ساختِ نصب‌کننده:
npm start
```

## نکته

پوشهٔ `app/` هنگام ساخت به‌صورت خودکار از `index.html` و دارایی‌های ریشهٔ ریپو پر می‌شود
(اسکریپت `scripts/prepare.js`) و در گیت نگه داشته نمی‌شود.
