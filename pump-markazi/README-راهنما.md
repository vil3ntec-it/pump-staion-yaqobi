# پمپ بنزین مرکزی — نسخهٔ مستقل

این یک نسخهٔ کاملاً مستقل از برنامهٔ مدیریت پمپ است (نسخه 2.9.57).
به هیچ سایت یا سرور دیگری وصل نیست — نه هاست می‌خواهد، نه دامنه، نه سرور.

## استفاده بدون اینترنت (همین حالا کار می‌کند)

فایل `index.html` را در مرورگر (کروم) باز کنید — همین.
تمام اطلاعات روی خودِ دستگاه (localStorage مرورگر) ذخیره می‌شود و برنامه
بدون اینترنت هم کامل کار می‌کند.

## فعال کردن سینک آنلاین (فایربیس جدا — رایگان)

اگر می‌خواهید اطلاعات بین چند دستگاه سینک شود، یک پروژهٔ رایگان فایربیس
مخصوص همین مشتری بسازید (به فایربیسِ هیچ پمپ دیگری وصل نیست):

1. به `console.firebase.google.com` بروید و با یک جیمیل وارد شوید.
2. **Add project** → یک اسم بدهید (مثلاً `pump-markazi`) → ادامه تا ساخته شود.
3. از منوی چپ: **Build → Realtime Database → Create Database** → حالت
   **test mode** را انتخاب کنید.
4. آدرس دیتابیس را کپی کنید (شبیه `https://....firebaseio.com`).
5. از **Project settings** (چرخ‌دنده) مقدار **Web API Key** و **Project ID** را بردارید.
6. در فایل `index.html` دنبال `FIREBASE_CONFIG` بگردید و سه مقدار را جایگزین کنید:

```js
const FIREBASE_CONFIG = {
  apiKey: "کلید API شما",
  databaseURL: "آدرس دیتابیس شما",
  projectId: "شناسه پروژه شما"
};
```

تا وقتی این سه مقدار پر نشود، برنامه فقط آفلاین کار می‌کند و به هیچ سروری
وصل نمی‌شود.

## نکته‌ها

- رمز ورود و بقیهٔ تنظیمات از داخل خود برنامه قابل تغییر است.
- اعلان بیرون از برنامه (ntfy) کد اشتراک مخصوص خودش را همان اول می‌سازد و
  با هیچ نصب دیگری قاطی نمی‌شود.
