# سرور شخصی پمپ یعقوبی (جایگزین فایربیس)

این پوشه یک سرور کوچک است که **جای فایربیس** را می‌گیرد: همهٔ داده‌ها، بکاپ‌ها، چت و «حضور آنلاین» به‌جای سرور گوگل، روی **کامپیوتر ویندوزی خودت** ذخیره می‌شوند. با این کار محدودیت فضای فایربیس برداشته می‌شود.

- **زبان:** Node.js
- **پایگاه‌داده:** PostgreSQL
- **ارتباط با برنامه:** WebSocket (بلادرنگ، مثل خود فایربیس)
- **کارِ آفلاین:** حفظ می‌شود؛ تغییراتِ آفلاین در صف می‌مانند و به‌محضِ وصل‌شدنِ نت و سرور، خودکار ذخیره می‌شوند.

> نکتهٔ مهم: این تغییر **اختیاری** است. تا وقتی آدرس سرور را در برنامه وارد نکنی، برنامه دقیقاً مثل قبل با فایربیس کار می‌کند.

---

## پیش‌نیازها (یک‌بار نصب روی ویندوز)

### ۱) نصب Node.js
از سایت <https://nodejs.org> نسخهٔ **LTS** را دانلود و نصب کن (نصبِ پیش‌فرض، فقط Next).
برای اطمینان، در Command Prompt بزن:
```
node --version
```
باید یک شماره نسخه (مثل `v20...`) نشان دهد.

### ۲) نصب PostgreSQL
از <https://www.postgresql.org/download/windows/> نصب‌کننده را بگیر و نصب کن.
هنگام نصب:
- یک **رمز** برای کاربر `postgres` می‌خواهد → یک رمز بگذار و **یادت بماند**.
- پورت پیش‌فرض `5432` را دست نزن.

بعد از نصب، برنامهٔ **pgAdmin** یا **SQL Shell (psql)** نصب می‌شود. یک دیتابیس بساز:

در «SQL Shell (psql)» باز کن، Enter بزن تا به مقادیر پیش‌فرض برسی، رمز `postgres` را بده، بعد:
```sql
CREATE DATABASE pump_yaqobi;
```

## راه‌اندازی سرور

### ۳) گرفتن فایل‌های سرور
کل پوشهٔ `server` را روی کامپیوتر خودت داشته باش (همین پوشه). داخلش Command Prompt باز کن.

### ۴) ساختن فایل تنظیمات `.env`
فایل `.env.example` را کپی کن و اسمش را بگذار `.env`، بعد بازش کن و پر کن:
```
PORT=8787
AUTH_TOKEN=یک-رمز-طولانی-و-تصادفی   ← این را بعداً در برنامه هم می‌گذاری
DATABASE_URL=postgresql://postgres:رمزِ‌پستگرس@localhost:5432/pump_yaqobi
```
- `AUTH_TOKEN` را یک رشتهٔ طولانی و سخت بگذار (مثلاً ۳۰ کاراکتر تصادفی). این رمزِ سرور توست.
- در `DATABASE_URL` به‌جای «رمزِ‌پستگرس» همان رمزی را بگذار که موقع نصب PostgreSQL انتخاب کردی.

### ۵) اجرا
ساده‌ترین راه: روی فایل **`start-windows.bat`** دوبار کلیک کن.
- بار اول خودش وابستگی‌ها را نصب می‌کند و اگر `.env` نبود می‌سازد.
- بعد سرور بالا می‌آید و پیام «✅ سرور پمپ یعقوبی روی پورت 8787 بالا آمد» را می‌بینی.

یا دستی در Command Prompt:
```
npm install
npm start
```

تست سلامت: در مرورگر برو به <http://localhost:8787/health> — باید `{"ok":true,...}` ببینی.

## در دسترس کردن از اینترنت (Cloudflare Tunnel — رایگان)

کامپیوتر خانگی IP ثابت ندارد و از بیرون مستقیم دیده نمی‌شود. با یک «تونل» رایگان، یک آدرس اینترنتی امن می‌گیری، **بدون** باز کردن پورت روی مودم.

### ۶) نصب cloudflared
از <https://github.com/cloudflare/cloudflared/releases> فایل `cloudflared-windows-amd64.exe` را بگیر، اسمش را بگذار `cloudflared.exe`.

### ۷) اجرای تونل
در Command Prompt، در پوشه‌ای که `cloudflared.exe` هست:
```
cloudflared tunnel --url http://localhost:8787
```
یک آدرس مثل زیر به تو می‌دهد:
```
https://random-words-1234.trycloudflare.com
```
این آدرس را بردار. **در برنامه، به‌جای `https` بنویس `wss`** (چون اتصال WebSocket است):
```
wss://random-words-1234.trycloudflare.com
```

> آدرسِ رایگانِ `trycloudflare.com` با هر بار اجرا عوض می‌شود. برای آدرسِ ثابت (مثلاً `pump.yourdomain.com`) باید یک دامنه در کلادفلر ثبت کنی و «Named Tunnel» بسازی — راهنمای کاملش در مستندات کلادفلر هست. برای شروع، همان آدرس رایگان کافی است.

## وصل کردن برنامه به سرور خودت

برنامه (سایت) را باز کن، سپس **کنسول مرورگر** را باز کن (کلید `F12` → تب Console) و این دو خط را بزن (آدرس و رمز خودت را بگذار):
```js
localStorage.setItem('pumpServerUrl',  'wss://random-words-1234.trycloudflare.com');
localStorage.setItem('pumpServerToken','همان AUTH_TOKEN که در .env گذاشتی');
location.reload();
```
از این لحظه برنامه به‌جای فایربیس به سرور تو وصل می‌شود. در کنسول باید ببینی:
```
[پمپ یعقوبی] حالت سرور شخصی فعال شد → wss://...
```

**برای برگشت به فایربیس** (اگر خواستی):
```js
localStorage.removeItem('pumpServerUrl');
localStorage.removeItem('pumpServerToken');
location.reload();
```

## نکته‌های مهم

- **کامپیوتر باید روشن باشد** تا کاربران به سرور دسترسی داشته باشند. اگر خاموش شود، تغییراتِ آفلاینِ کاربران در دستگاه خودشان می‌ماند و **به‌محض روشن‌شدنِ دوبارهٔ سرور و وصلِ نت، خودکار ذخیره می‌شود** (هیچ داده‌ای گم نمی‌شود، فقط با تأخیر می‌رود).
- **بکاپ بگیر.** داده در PostgreSQL است. برای بکاپ دستی:
  ```
  pg_dump -U postgres pump_yaqobi > backup.sql
  ```
  برگرداندن:
  ```
  psql -U postgres pump_yaqobi < backup.sql
  ```
- **امنیت:** `AUTH_TOKEN` را به کسی نده و در گیت‌هاب نگذار (فایل `.env` در `.gitignore` هست). چون داده مالی است، رمز قوی بگذار.
- **اجرای همیشگی:** اگر می‌خواهی سرور با روشن‌شدن ویندوز خودکار بالا بیاید، می‌توانی `start-windows.bat` را در پوشهٔ Startup ویندوز بگذاری، یا از ابزار `pm2` / «Task Scheduler» استفاده کنی.

با سرورِ روشن، در همین پوشه:
```
node test-client.mjs
```
اگر همه‌چیز درست باشد، ۱۲ تستِ سبز می‌بینی (set/get/چت/حضور آنلاین و ...).
